# Guide d'utilisation des liens en HTML

## Introduction
En HTML, l'élément `<a>` est utilisé pour créer des liens hypertexte, permettant aux utilisateurs de naviguer vers d'autres pages web, fichiers ou sections d'une même page.

## Syntaxe de base
Voici comment créer un lien en HTML :

```html
<a href="https://www.example.com">Cliquez ici</a>

sinon tu mets dans le href le dossier et le fichier vers lequel tu veux pointer : 

<a href="PortefeuillePreuves/TrucMuche.pdf">Cliquez ici</a>


¨POUR UN BOUTON

<button>Cliquez-moi</button>
